## ğŸ› **Troubleshooting**

### **Guests not loading?**
- Check your Firebase configuration in `firebase.ts`
- Ensure Firestore rules allow read/write to `guestPayments` collection
- Check browser console for errors

### **Can't add guests?**
- Ensure all required fields (marked with *) are filled
- Check that amount is a valid number
- Verify Firebase write permissions

### **Totals not calculating correctly?**
- Verify the exchange rate in `guestPayment.ts`
- Check that amounts are being saved as numbers, not strings
